# 広島県内雨量可視＋

広島県内の雨量観測局データをリアルタイムで可視化するWebアプリケーションです。

## 機能

- 広島県全域の雨量観測局をインタラクティブな地図上に表示
- 60分降水量のリアルタイムアニメーション再生
- 任意期間における **最大60分降水量** / **最大24時間降水量** の評価
- 国の災害採択基準（24h 80mm / 1h 20mm）超過局のハイライト表示
- 採択基準超過局リストのCSV出力
- 薄板スプライン補間（TPS）による等雨量線の自動描画

## データソース

広島県防災情報ポータル  
`https://www.bousai.pref.hiroshima.lg.jp/`

## セットアップ（ローカル実行）

```bash
# 依存パッケージのインストール
npm install

# サーバー起動
npm start

# ブラウザでアクセス
open http://localhost:5500
```

## 使い方

1. **期間設定** で分析したい日付範囲を選択
2. **データ更新** ボタンをクリックして広島県サイトからデータを取得
3. **表示モード** を切り替えてリアルタイム / 最大60分 / 最大24時間降水量を確認
4. 採択基準超過局は右パネルに自動表示、CSVでエクスポート可能

## ディレクトリ構成

```
hiroshima-rainfall/
├── index.html            # フロントエンド（エントリポイント）
├── main.js               # 地図描画・補間ロジック
├── styles.css            # スタイル
├── process_data.js       # データ取得・処理スクリプト
├── mapping.json          # 観測局座標マッピング
├── rainfallObservatory.json  # 観測局マスタデータ
├── rainfall_data.json    # 処理済み雨量データ（自動生成）
├── 県内雨量局.csv          # 観測局一覧CSV
├── server/
│   └── server.js         # Express サーバー
├── tools/                # データ整備用ユーティリティ
└── backups/              # バックアップファイル
```

## 技術スタック

- **フロントエンド**: Vanilla JS / Leaflet.js / Turf.js
- **バックエンド**: Node.js / Express
- **データ処理**: xlsx（Excelパース）/ axios（HTTPダウンロード）
- **補間アルゴリズム**: 薄板スプライン（TPS）+ マーチングスクエア法
